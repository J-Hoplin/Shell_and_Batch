### VSCode C/C++ 설정파일 초기화 batch파일

- 이 배치파일 실행조건은 아래와 같습니다

  1. MinGW GCC/G++컴파일러가 C드라이브 최상위 디렉토리에(C:\) 설치되어있음 
  2. MinGW 컴파일러가 환경변수 등록이 되어있다는 가정

  * 만약 저와 MinGW 디렉토리가 다른 경우에는 이 파일의 vscode디렉토리 내의 설정파일을 자신의 설정파일을 복사해 넣어주시면 됩니다

- 만들게 된 이유는 VS Code에서 C/C++환경을 그때그떄 초기화 시켜 재설정하는일이 없도록 하기 위함입니다. 저같은 경우 이 배치파일 디렉토리도 환경변수에 등록하여 'cinit'이라는 명령어로 초기화
  될 수 있게끔 하여 쓰고있습니다.
